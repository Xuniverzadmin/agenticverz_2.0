| `/precus/incidents` | Incidents > Events > ACTIVE |

| `/precus/connectivity` | Connectivity > Integrations > SDK_SETUP |
| `/precus/account` | Account > Profile > OVERVIEW |

**Navigation Behavior:**
- Clicking **domain** in sidebar â†’ loads default subdomain + first topic
- Clicking **subdomain** in sidebar â†’ loads first topic of that subdomain
- Clicking **topic tab** â†’ switches topic within current subdomain
- State preserved in URL query params if needed (e.g., `?topic=completed`)
