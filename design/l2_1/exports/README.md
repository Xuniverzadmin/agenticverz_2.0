# AURORA_L2 Exports Directory

**Status:** SCAFFOLD (empty placeholder)

## Purpose

This directory will contain generated export artifacts from the AURORA_L2 pipeline.

## Export Types

| Export | Format | Purpose |
|--------|--------|---------|
| `ui_projection_lock.json` | JSON | Frozen UI projection (backward compatible) |
| `intent_store_seed.sql` | SQL | Seed data for intent_store table |
| `capability_binding.json` | JSON | Panel → Capability → Permission map |

## Generation

Exports are generated by running the AURORA_L2 compiler pipeline.
